# Bangalore-South-Crime-Detection-Analysis
	This report presents a data analytics case study developed using R programming as part of the curriculum, focusing on crime detection and analysis. The dataset contains records of crime incidents, including attributes such as crime type, location, time, and demographic information. The data, originally sourced from publicly available crime records, is cleaned and converted into a structured format excel for analysis in R.
	Key analytical techniques employed include descriptive statistics, data exploration, preprocessing, and predictive modeling, which provide a deeper understanding of crime patterns and possible risk factors. A simple predictive model is built to detect and classify potential crime incidents, supported by various visualizations and charts. This work demonstrates the practical application of data analytics methodologies in R and provides insights into crime prevention and public safety.
Tools : R ,RStudio, Excel
Source of dataset:
https://www.kaggle.com/datasets/vineetsingh26/bangalore-south-crime-details
